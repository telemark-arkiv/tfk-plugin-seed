# tfk-plugin-seed
Seed for creating plug-ins for tfk.

Comes with CRUD routes and handlers and tests.

Use this as starting point when developing plug-ins for Tfk.

# Installation
```
$ git clone git@github.com:telemark/tfk-plugin-seed.git
```

# Setup
```
$ npm run setup
```

# Start server
```
$ npm start
```

# Test
```
$ npm test
```
